# Kaggle Amazon Sales & Trading Insights Notebooks

These notebooks are designed to run on Kaggle with the dataset:
https://www.kaggle.com/datasets/nalisha/amazon-sales-and-trading-insights-dataset-2024

Each notebook:
- loads the raw dataset automatically,
- generates the requested research-question outputs,
- saves figures as PDF files in `outputs/`,
- saves tables as CSV files in `outputs/`.

Notebook mapping:
- RQ1_Baseline_Performance.ipynb
- RQ2_Model_Comparison.ipynb
- RQ3_Preprocessing_Effect.ipynb
- RQ4_Feature_Importance.ipynb
- RQ5_Metric_Sensitivity.ipynb
- RQ6_Robustness_Analysis.ipynb
- RQ7_Final_Recommendation.ipynb
